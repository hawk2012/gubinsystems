# GubinNET

**Собери свой сайт. По-настоящему.**

GubinNET — это модульная CMS для Node.js, созданная для разработчиков, которые ценят контроль и производительность. С темной админкой "Carbon Dark", простой установкой через npx и философией "инструмент, а не игрушка", она позволяет собирать сайты из необходимых модулей (магазин, блог, SEO), оставляя полную свободу в создании фронтенда.

## Философия

GubinNET построен на трех китах:

- **Предсказуемость**: Никакой "магии", код должен быть понятным и логичным.
- **Контроль**: Разработчик всегда знает, где и что происходит. Минимум скрытых процессов.
- **Производительность**: Быстрая из коробки, без лишнего наворотов.

## Установка

```bash
npx create-gubinet-app my-site
```

## Архитектура

GubinNET использует архитектуру "Ядро + Модули":

- **Ядро (Core Gubin)**: Минималистичное ядро с базовыми функциями
- **Модули**: Независимые npm-пакеты, которые подключаются к ядру
  - `@gubinet/module-blog` — Блог с тегами, категориями, комментариями
  - `@gubinet/module-shop` — Магазин (корзина, товары, категории, заказы)
  - `@gubinet/module-seo` — Карта сайта, мета-теги, управление robots.txt
  - `@gubinet/module-pages` — Управление страницами

## Технический стек

- **Backend**: Node.js + Fastify
- **База данных**: Prisma ORM с поддержкой PostgreSQL, SQLite, MySQL
- **Админка**: Отдельное SPA-приложение с темной темой
- **Фронтенд**: Полная свобода выбора фреймворка (Next.js, Nuxt, Astro и т.д.)

## Особенности

- Честная документация с примерами кода
- Производительность как фича

# Михаил Губин - DevOps/SRE-инженер

Официальный сайт Михаила Губина - DevOps/SRE-инженера с опытом 9 лет. Специализация на стабильности, безопасности и автоматизации.

## Структура проекта

- `index.html` - Главная страница сайта
- `/css/` - Стили сайта
- `/js/` - JavaScript файлы
- `/pages/` - Статические страницы (о себе, навыки, опыт и т.д.)
- `/blog/` - Страницы блога
- `/_posts/` - Markdown-файлы постов для Jekyll-блога
- `/_layouts/` - Шаблоны для Jekyll
- `/_config.yml` - Конфигурационный файл Jekyll

## Особенности

- **Статический сайт** - Основные страницы сайта являются статическими HTML-файлами
- **Динамический блог** - Блог реализован с помощью Jekyll, что позволяет GitHub Pages динамически генерировать страницы блога
- **SEO-оптимизация** - Все страницы содержат необходимые мета-теги, schema.org разметку и канонические ссылки
- **Адаптивный дизайн** - Сайт корректно отображается на устройствах с разными размерами экрана
- **Навигация** - Удобное меню навигации на всех страницах

## Блог

Блог реализован с использованием Jekyll, что позволяет:

- Создавать новые посты в формате Markdown
- Автоматически генерировать страницы постов
- Использовать шаблоны для оформления
- Иметь динамическую ленту постов на главной странице блога

## GitHub Pages

Сайт развернут на GitHub Pages и включает в себя:

- CNAME файл для кастомного домена (gubin.systems)
- Статические HTML-страницы для основного контента
- Jekyll-совместимую структуру для динамического блога
- Полностью SEO-оптимизированный контент

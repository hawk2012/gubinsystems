# Михаил Губин - DevOps в Финтехе

Официальный сайт Михаила Губина - DevOps/SRE-инженера с опытом 9 лет. Специализация на стабильности, безопасности и автоматизации в финансовых структурах.

## Описание

Этот сайт представляет собой многостраничное веб-приложение, построенное с использованием Node.js и Express. Он включает в себя следующие разделы:

- **Главная страница**: Общая информация о профессиональной деятельности
- **Опыт работы**: Подробное описание карьерного пути и достижений
- **Навыки**: Технические компетенции и опыт работы с различными инструментами
- **Контакты**: Контактная информация и форма обратной связи

## Особенности

- **Многостраничный интерфейс** с навигацией слева
- **Профессиональный дизайн** в деловом стиле
- **Двухколоночная верстка** с меню слева
- **Часы для Москвы и Ташкента** в боковом меню
- **Динамический год в футере** (2025-2026)
- **Подробные технические описания** на всех страницах
- **Адаптивный дизайн** для различных устройств

## Технологии

- Node.js
- Express.js
- EJS (Embedded JavaScript templating)
- HTML5
- CSS3
- JavaScript (ES6+)

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

3. Откройте браузер и перейдите по адресу: http://localhost:3000

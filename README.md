# DevOps Professional Portfolio

## Overview
This is a modern, cloud-themed portfolio website for a DevOps engineer. The design features a clean, professional look with interactive elements that showcase DevOps skills and experience.

## Features
- Responsive design that works on all devices
- Modern cloud-themed UI with animations
- Interactive navigation and smooth scrolling
- Skills section with animated progress bars
- Timeline for work experience
- Contact form with validation
- Mobile-friendly hamburger menu

## Design Elements
- Cloud and server rack visualizations
- Animated network pulse effects
- Code block display with terminal-like styling
- Skill progress bars with percentage indicators
- Technology icons with hover effects
- Timeline with alternating layout

## Technologies Used
- HTML5
- CSS3 (with custom properties and animations)
- JavaScript (with modern ES6+ features)
- Font Awesome for icons
- Google Fonts for typography

## Sections
1. **Hero**: Introduction with call-to-action buttons
2. **About**: Personal information and statistics
3. **Skills**: Technical skills with progress indicators
4. **Experience**: Work history in timeline format
5. **Contact**: Contact information and form

## Color Scheme
- Primary: #4361ee (blue)
- Secondary: #7209b7 (purple)
- Accent: #4cc9f0 (light blue)
- Success: #4ade80 (green)
- Warning: #facc15 (yellow)
- Danger: #f87171 (red)

## Responsive Design
The site is fully responsive and adapts to different screen sizes:
- Desktop: Full layout with all elements visible
- Tablet: Adjusted grid layouts and spacing
- Mobile: Stacked elements and mobile navigation

## JavaScript Functionality
- Mobile menu toggle
- Smooth scrolling navigation
- Animated skill bars on scroll
- Form validation and submission
- Scroll-based animations
- Active navigation highlighting
- Button ripple effects

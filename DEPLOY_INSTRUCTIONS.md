# Инструкция по деплою на GitHub Pages

## Подготовка репозитория

1. Создайте новый репозиторий на GitHub с именем `[REPO_NAME]` (или используйте существующий)
2. Инициализируйте Git в текущей папке (если еще не сделано):
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```

3. Свяжите локальный репозиторий с удаленным:
   ```bash
   git remote add origin https://github.com/[YOUR_GITHUB_USERNAME]/[REPO_NAME].git
   ```

4. Отправьте код в репозиторий:
   ```bash
   git push -u origin main
   ```

## Настройка GitHub Pages

1. Перейдите в настройки вашего репозитория на GitHub
2. Найдите раздел "Pages" в левой панели
3. В настройках Source выберите ветку `gh-pages` и папку `/ (root)`
4. Сохраните изменения

## Деплой приложения

1. Убедитесь, что вы заменили `[YOUR_GITHUB_USERNAME]` и `[REPO_NAME]` на реальные значения в файле `package.json`
2. Выполните команду деплоя:
   ```bash
   npm run deploy
   ```

Эта команда:
- Выполнит предварительную сборку (`npm run build`)
- Создаст статическую версию сайта в папке `dist`
- Отправит содержимое папки `dist` в ветку `gh-pages` вашего репозитория

## После деплоя

После успешного деплоя ваш сайт будет доступен по адресу:
`https://[YOUR_GITHUB_USERNAME].github.io/[REPO_NAME]`

## Важные замечания

1. Приложение было преобразовано из динамического Node.js-приложения в статическую версию для совместимости с GitHub Pages
2. Все страницы теперь представлены в виде HTML-файлов
3. Все маршруты преобразованы в статические HTML-файлы (например, `/experience` станет `experience.html`)
4. Статические ресурсы (CSS, JS, изображения, PDF) находятся в папке `assets`

## Возможные проблемы и решения

1. Если сайт не отображается после деплоя:
   - Проверьте, правильно ли настроена ветка gh-pages в настройках GitHub
   - Отключите и снова включите GitHub Pages в настройках репозитория

2. Если пути к ресурсам неправильные:
   - Убедитесь, что вы правильно заменили плейсхолдеры в поле homepage
   - Проверьте, что все ссылки в HTML-файлах указывают на правильные пути

# Mikhail Gubin - DevOps/SRE Engineer Portfolio

This is a professional Jekyll-based portfolio website for Mikhail Gubin, a DevOps/SRE engineer with 9+ years of experience in infrastructure stability, automation, and security.

## Site Structure

The site includes the following pages:

- **Home** (`index.md`) - Main landing page with overview
- **About** (`about.md`) - Personal statement and professional approach
- **Experience** (`experience.md`) - Detailed work history
- **Skills** (`skills.md`) - Technical expertise and competencies
- **Education** (`education.md`) - Educational background
- **Contact** (`contact.md`) - Contact information

## Technical Implementation

- Jekyll configuration in `_config.yml`
- Custom layouts in `_layouts/` directory
- Custom styling in `assets/css/style.css`
- Responsive design for all devices

## How to Run

```bash
# Install Jekyll
gem install jekyll bundler

# Run the site
bundle exec jekyll serve
```
